standard_development.json
=========================

This configuration file is used by the for acceptance test: AcceptanceTests_other.js.

It includes to the basic localInstall setup for acceptance tests (modifying the preferences folder used), which in turn includes the standard development config file (running GPII locally, using development setup). It makes no specific changes to the configuration other than those in the mentioned files - this also means the device reporter folder is the default and NOT specific to the acceptance test.
